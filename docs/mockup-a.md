# モックアップ A: ヒーロー + プロジェクト一覧

シンプルに自己紹介 → プロジェクトが並ぶ構成。

```
┌─────────────────────────────────────────────┐
│  [ロゴ]  Projects   Profile   Blog     [■]  │  ← ヘッダー (ナビ)
├─────────────────────────────────────────────┤
│                                             │
│          Red → Green → Refactor             │  ← キャッチコピー
│                                             │
│    テスト駆動開発で作るプロダクトたち。          │  ← サブテキスト
│    まだ赤いリンクは、これからのテスト。          │
│                                             │
├─────────────────────────────────────────────┤
│                                             │
│  Projects                                   │
│                                             │
│  ┌───────────┐ ┌───────────┐ ┌───────────┐  │
│  │ 🔴        │ │ 🟢        │ │ 🔵        │  │
│  │ my-blog   │ │ tdd-calc  │ │ api-kit   │  │
│  │           │ │           │ │           │  │
│  │ 進行中     │ │ 完成      │ │ 改善済み   │  │
│  │ Haskell   │ │ Haskell   │ │ Go        │  │
│  └───────────┘ └───────────┘ └───────────┘  │
│                                             │
├─────────────────────────────────────────────┤
│  © 2026 suzukiwataru                        │  ← フッター
└─────────────────────────────────────────────┘
```

- ヒーローセクションでTDDの世界観を伝える
- プロジェクトカードの左上にステータスの色（赤/緑/青）
- カードのリンク色も未訪問=赤、訪問済み=緑、更新あり=青
